# Digibord dashboard, publicatie via GitHub Pages

Repository naam: **digibord**

Snelle publicatie:
1. Upload deze projectmap naar een GitHub repository die **digibord** heet
2. Ga naar Settings, Pages
3. Source: GitHub Actions
4. Doe een commit, of push naar de branch **main**
5. Wacht tot de workflow klaar is, daarna staat je site live op GitHub Pages

Lokaal testen:
1. npm install
2. npm run dev

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
